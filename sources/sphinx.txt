=========================
Using Sphinx
=========================

Installing into MacOSX 10.7 (Lion)
==================================

Run "sudo easy_install pip". And then, run "sudo pip install Sphinx".
That's all.

How to create tach.github.com
=============================

1. create github repository: https://github.com/tach/tach.github.com
2. clone the repository and create "sources" directory
3. run "sphinx-quickstart" command on "sources" directory
4. change your Makefile like https://github.com/tach/tach.github.com/sources/Makefile

* https://github.com/michaeljones/sphinx-to-github
